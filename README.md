# js-practice-problems-01
practice problems to demonstrate your skills


Welcome to your first interactive JavaScript lab! You will be solving **10 logic problems** and then viewing the results on your very own live website.

## 🚀 Quick Start (Browser Workflow)

1.  **Open the Editor**: Click on the file named `assignment.js` above.
2.  **Enter Code Mode**: Press the **`.` (period)** key on your keyboard to open the Web Editor.
3.  **Write Your Logic**: Find the functions labeled `// TODO` and write your code between the curly braces `{ }`.
4.  **Save & Submit**: 
    * Click the **Source Control** icon (the branch symbol) on the left.
    * Type a message like "Finished Task 1".
    * Click **Commit & Push**.

---

## 📝 The 10 Tasks

Your goal is to complete these functions inside `assignment.js`:

| Category | Tasks |
| :--- | :--- |
| **Logic & Loops** | `sumArray`, `filterHighNumbers`, `fizzBuzz`, `findMax` |
| **Arrays & Objects** | `getStringLengths`, `createUser`, `getAdultNames`, `titleCase` |
| **Advanced** | `countChar`, `isPalindrome` |

## 🌐 How to View Your Live Website

Once you have pushed your code, you can see your results on a real webpage:

1.  Click the **Settings** tab at the top of this GitHub page.
2.  On the left menu, click **Pages**.
3.  Under **Branch**, change "None" to **`main`**.
4.  Click **Save**.
5.  **Wait 60 seconds.** Refresh the page. A link will appear: *"Your site is live at..."*
6.  Click that link to see your "JS Assignment Dashboard"!

## 🧪 How is this Graded?

This assignment is **auto-graded**. Every time you "Commit & Push," GitHub runs a series of hidden tests.

* **Check your score**: Look at the **Actions** tab at the top of the screen.
* **Green ✅**: Your logic is perfect!
* **Red ❌**: Something is wrong. Click on the failed "Autograding" run to see exactly which problem failed and why.

## 💡 Pro-Tips for Success
* **Case Sensitivity**: In Task 3 (FizzBuzz), make sure you return "Fizz" exactly (capital F), not "fizz".
* **Objects**: In Task 6, ensure your object keys are exactly `name` and `age`.
* **The Console**: When viewing your live website, press **F12** (or Right-Click > Inspect) and go to the **Console** tab to see helpful debug messages.
